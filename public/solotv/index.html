How it works

SoLoTV ships repository.solotv and a setup wizard that installs the SoLoTV Build Wizard. That wizard installs the Xenon interface and streaming add-ons. The catalog is the same family as Xenon; the repo name and wizard labels are SoLo-branded.

Install

  1. Install repository.solokodi, then SoLoKodi Setup.
  2. In setup, choose Switch to SoLoTV and run the wizard.
  3. Or add file source https://solokodi.sololink.cloud/solotv/ and install repository.solotv from ZIP.
