SoLoTV

A SoLo-branded streaming build on top of Diggz Xenon — movies, series, live TV, and Debrid-ready addons, with our setup wizard and update flow.

Download SoLoKodi Repo Build Manifest SoLoKodi Kids

Powered by Diggz

SoLoTV installs the official Diggz repository and Chef Omega Wizard, then you pick Xenon 4K (Debrid or Free) inside the Chef wizard — the same stack used by millions of Kodi users.

SoLo layer

  • SoLoTV Setup Wizard — Diggz repo, Chef wizard, favourites, optional Real-Debrid
  • SoLo blue accent — branded Kodi colors until Xenon applies its skin
  • Update Build Now — sync SoLoKodi repo and repair shortcuts
  • Switch builds — move between SoLoTV and SoLoKodi Kids from setup

Install

  1. Install repository.solokodi from the ZIP above.
  2. Install SoLoKodi Setup from the SoLoKodi repository.
  3. Open setup and choose Switch to SoLoTV, then run the wizard.
  4. In Chef Omega Wizard, install Xenon 4K and restart Kodi.
