SoLoKids TV

A kid-friendly clone of SoLoTV with bright artwork, curated kids Trakt lists, simple streaming shortcuts, and Real-Debrid-ready playback.

SoLoKodi Repo Build List SoLoTV

Install

  1. Install repository.solokodi, then install SoLoKodi Setup.
  2. Choose SoLoKids TV in the build picker.
  3. Connect Real-Debrid, then open SoLoKids TV Build Wizard.
  4. Choose SoLoKids TV 4K and restart Kodi after the interface installs.

Profile

SoLoKids TV is separate from SoLoKodi Kids. It does not install the official kids-source menu set. Kids Movies and Kids TV open curated public Trakt lists, while playback is expected to run through Real-Debrid-capable streaming add-ons in the restored build.
