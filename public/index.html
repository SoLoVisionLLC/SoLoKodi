SoLoVision Kodi Build

SoLoKodi Family Build

A calm, locked-down Kodi setup for kids, with official family add-ons and parent-managed Real-Debrid authorization for lawful personal media.

Download Kodi Repo ZIP Repository Feed

Kid Surface

Run the setup add-on from Kodi to enable PBS Kids, curated YouTube access, and family favourites without exposing settings or source editing.

Parent Lock

The setup add-on opens a lock checklist for profiles, Master Lock, add-ons, file manager, and approved media sources.

Real-Debrid

Real-Debrid uses device authorization. Credentials are created by the parent and stored only in the local Kodi profile.

Install

  1. In Kodi, enable installing from unknown sources only long enough to install this repository ZIP.
  2. Install repository.solokodi from the downloaded ZIP.
  3. Open Install from repository, choose SoLoKodi, and install SoLoKodi Family Setup.
  4. Run Family Setup, then apply the parent lock checklist.

Legal Boundary

This build does not include third-party piracy repositories, scraper packs, shared Real-Debrid credentials, or torrent search. Real-Debrid support is for lawful personal media access only.
