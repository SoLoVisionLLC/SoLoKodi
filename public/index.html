Install

  1. In Kodi, enable installing from unknown sources only long enough to install this repository ZIP.
  2. Install repository.solokodi from the downloaded ZIP.
  3. Open Install from repository, choose SoLoKodi, and install SoLoKodi Kids Setup.
  4. Run the Setup Wizard and restart Kodi when prompted.
  5. Use Check for Updates anytime to stay on the latest build.
